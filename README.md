# SOGHeatSeeker DDoS Tool

A Python-based DDoS simulation tool for testing network resilience against denial-of-service attacks.

## Features

- Simulated TCP flood attack
- Adjustable packet size and thread count
- Real-time attack status monitoring
- Packet logging
- Stop/start functionality
- Thread-safe operation

## Requirements

- Python 3.x
- Tkinter (usually included with Python)

## Installation

```bash
pip install -r requirements.txt
